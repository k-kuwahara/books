コンストラクタ

「Functionって誰？」となると思いますが、この子はJavaScriptに元々定義されたオブジェクト（厳密にはコンストラクタ、つまり関数）です。 他にはObject, Number, Array, Booleanなどがいます。

この子らはwindowオブジェクトに定義されているため、プログラムのどこからでも呼べます。
